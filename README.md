Atividade – Aplicação de Orientação a Objetos em Java

Este repositório contém uma atividade prática desenvolvida em Java com o objetivo de aplicar os conceitos básicos de Programação Orientada a Objetos (POO).

O projeto foi criado como exercício de estudo para praticar a modelagem de classes, criação de objetos e organização do código utilizando os princípios da orientação a objetos, como encapsulamento e separação de responsabilidades. A POO é um paradigma amplamente utilizado no desenvolvimento de software moderno para estruturar aplicações de forma mais organizada e reutilizável. ()

Objetivo

Praticar conceitos fundamentais de Java e reforçar o entendimento sobre:

Classes e objetos

Atributos e métodos

Organização de código em projetos orientados a objetos

Tecnologias

Java
